# Clean Max II Auto Detail

Static single-page website for Clean Max II Auto Detail in Houston, Texas.

## What is included

- Responsive storefront-inspired homepage
- Services and package pricing menu
- Quote request form
- Gallery placeholders for detailing results
- Fleet and dealer detailing section
- Reviews, location, hours, and footer
- Sticky mobile action bar for call, services, and directions

## Local preview

Open `index.html` in a browser. No build step is required.
